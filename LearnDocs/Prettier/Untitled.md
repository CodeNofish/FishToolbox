# About

## What is Prettier?

Prettier 是一个固执己见的代码格式化程序，支持：

* JS
* JSX
* Angular
* Vue
* Flow
* TypeScript
* CSS Less and SCSS
* HTML
* Ember/Handlebars
* JSON
* GraphQL
* Markdown
* YAML

它删除了所有原始样式*并确保所有输出的代码符合一致的样式。 （参见这篇博文）

Prettier 获取您的代码并考虑行长度，从头开始重新打印它。
