# cache

> Canary
>
> 缓存仅适用于 React 服务器组件。请参阅支持 React 服务器组件的框架。
>
> 缓存仅在 React 的 Canary 和实验通道中可用。在生产中使用缓存之前，请确保您了解限制。在此处了解有关 React 发布渠道的更多信息。

`cache` lets you cache the result of a data fetch or computation.

缓存允许您缓存数据获取或计算的结果。

```react
const cachedFn = cache(fn);
```
